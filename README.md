# 📌 KeenKeeper

## 📖 Description
KeenKeeper is a Contact management web application designed to help users maintain meaningful connections with friends and family. It provides a simple, structured way to track interactions, monitor relationship health, and stay consistent with communication through reminders and status indicators.

---

## 🚀 Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- React.js  
- Context API  

---

## ✨ Key Features
- 📊 **Statistics Tracking Dashboard**

- ⏰ **Interaction Monitoring System** 

- 👥 **Dynamic Friend Management**
